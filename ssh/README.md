# SSH Project

## Task 0
Script `0-use_a_private_key` connects to the server using the private key `school`.

## Task 1
Script `1-create_ssh_key_pair` generates an RSA key pair named `school` with passphrase `betty`.

## Task 2
SSH client config `2-ssh_config` allows passwordless login using the project key.

## Task 3
The public key `school.pub` has been added to `authorized_keys` on the server to allow ALU reviewers to log in.

## Notes
- Project key is separate from personal keys.
- Scripts are executable.
